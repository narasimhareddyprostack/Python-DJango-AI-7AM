it used to identify a specific resouces.

What is Path Parameter?
Path parameter is variable it is part of url.

https://www.flipkart.com/big-billion-days-store?Param=40003556665
https://www.flipkart.com/
big-billion-days-store                - Path Parameter
?Param=40003556665                    - Query Parameter

Path Parameter : is variable part of url
Query Parameters:is value passed in url after ? symbol

When to use Query Parameters?
you want to filter, sort, pagination or modify result.


URL
/products?category=beauty       -search beauty products
/products?page=2&limit=20       -pagination
/products?sort=price_desc       -sorting
